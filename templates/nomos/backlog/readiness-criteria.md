## Quando uma Demanda Está Pronta Para Virar Especificação

Verifique se:

- o problema está claro;
- o objetivo está claro;
- a dependência principal é conhecida;
- os riscos críticos foram identificados;
- as perguntas abertas que realmente bloqueiam foram respondidas;
- existe justificativa para prioridade e ordem de construção.

## Checklist

- [ ] Problema definido
- [ ] Escopo inicial delimitado
- [ ] Dependências conhecidas
- [ ] Riscos principais registrados
- [ ] Critério de sucesso definido
